1- Copy the request from ex02-rendu.txt into Kibana and run all the request

2- Run the following request in Kibana and check you got only 2 hits of tweet posted in 2016 and beyond:
GET twitter_better_mapping/_search
{
  "query": {
    "range": {
      "post_date": {
        "gte": "01 01 2016"
      }
    }
  }
}

3- Run the following command and check you got only one hit from user 'mimich':
GET twitter_better_mapping/_search
{
  "query": {
    "match": {
      "user": "mimich"
    }
  }
}

4- Run the following command:
POST twitter_better_mapping/_doc
{
  "test_filed": "plop"
}

{
  "error": {
    "root_cause": [
      {
        "type": "strict_dynamic_mapping_exception",
        "reason": "mapping set to strict, dynamic introduction of [test_filed] within [_doc] is not allowed"
      }
    ],
    "type": "strict_dynamic_mapping_exception",
    "reason": "mapping set to strict, dynamic introduction of [test_filed] within [_doc] is not allowed"
  },
  "status": 400
}
